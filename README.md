# KaKaoSearch
KaKao Image Search API 를 이용한 이미지 검색 앱 만들기

![image](https://user-images.githubusercontent.com/50766393/122225727-4761d980-cef0-11eb-934c-6d1ffdc705ab.png)

사용하려면 별도의 개인 API를 사용해야 함

## 사용한 라이브러리
* Android Architecture Components
  * LiveData
  * Lifecycle
  * ViewModel
  * ViewBinding
  * Navigation
  * Safe Args
  * Dagger Hilt
  * Paging 3

* UI
  * ConstraintLayout
  * RecyclerView
  * Fragment
  * ProgressBar
  * SearchView

* Third Party
  * Kotlin Coroutine
  * Glide
  * Okhttp3
  * Gson
  * Retrofit
  * KaKaoAPI
